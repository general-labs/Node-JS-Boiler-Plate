Node JS Boiler Plate
---------------

```sh
# Install dependencies
npm install

# Start development live-reload server
PORT=8080 npm run dev

# Start production server:
PORT=8080 npm start


OR

docker-compose up -d
```

http://localhost:4000/

http://localhost:4000/another_route
